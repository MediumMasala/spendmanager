# Data Safety Declaration (Google Play)

## Overview

This document contains the information needed to complete Google Play's Data Safety form.

## Data Collection Summary

| Category | Data Type | Collected | Shared | Purpose |
|----------|-----------|-----------|--------|---------|
| Personal info | Phone number | Yes | No | Account management |
| Financial info | Purchase history | Yes | No | App functionality |
| Financial info | Other financial info | Yes | No | App functionality |
| App activity | App interactions | Yes | No | Analytics |
| Device or other IDs | Device ID | Yes | No | App functionality |

## Detailed Declaration

### Phone Number
- **Collected**: Yes
- **Shared**: No
- **Purpose**: Account management, authentication
- **Optional**: No (required for account)
- **Processed ephemerally**: No (stored as hash)
- **User can request deletion**: Yes

### Purchase/Transaction History
- **Collected**: Yes
- **Shared**: No
- **Purpose**: App functionality (core feature)
- **Optional**: No (app's primary function)
- **Processed ephemerally**: No
- **User can request deletion**: Yes

### Device Identifiers
- **Collected**: Yes
- **Shared**: No
- **Purpose**: App functionality (device management)
- **Optional**: No
- **Processed ephemerally**: No
- **User can request deletion**: Yes

## Security Practices

### Data Encryption
- ✅ Data encrypted in transit
- ✅ Data encrypted at rest (server-side)

### Data Deletion
- ✅ Users can request data deletion
- ✅ Data deleted within 30 days of request

## Data Handling

### Data Not Collected
- ❌ Location
- ❌ Contacts
- ❌ Photos/Videos
- ❌ Audio
- ❌ Files and docs
- ❌ Calendar
- ❌ SMS (Play Store version)
- ❌ Call logs
- ❌ Health info
- ❌ Email addresses

### Third-Party Data Sharing
- **Not shared with any third parties for advertising**
- **Not shared with any third parties for analytics**
- **Not shared with any data brokers**

## Independent Security Review
- Status: Not yet completed
- Plan: Will pursue after launch

## Notes for Form

When filling out the Data Safety form:

1. **Does your app collect or share any user data?**
   - Yes

2. **Is all user data encrypted in transit?**
   - Yes

3. **Do you provide a way for users to request data deletion?**
   - Yes

4. **Have you committed to follow the Play Families Policy?**
   - N/A (not a family app)

5. **Data types collected**:
   - Phone number (Account management)
   - Financial info - Purchase history (App functionality)
   - Device IDs (App functionality)

6. **For each data type, is it**:
   - Shared: No
   - Collected: Yes
   - Optional: Phone number is required
   - User can request deletion: Yes
